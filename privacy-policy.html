Privacy Policy

Last Updated: March 23, 2025

Introduction

This Privacy Policy describes how ElementForge ("we", "our", or "extension") collects, uses, and handles your information when you use our browser extension. ElementForge is designed to extract and transform website elements into HTML and CSS code while respecting your privacy.

Data Collection

ElementForge collects the following limited data:

  • Website Content: When you select elements on a webpage, ElementForge temporarily processes the HTML structure and CSS styling of those elements. This happens only when you explicitly choose to extract elements.
  • User Activity: ElementForge records your interactions with the extension interface, such as which extraction options you select.
  • User Preferences: We store your selected extraction settings locally in your browser to provide a consistent experience.

Data Usage

All data collected by ElementForge is:

  • Used solely for providing the core functionality of the extension
  • Processed entirely within your browser
  • Never transmitted to any external servers or third parties
  • Temporary, except for your preferences which are stored locally

Data Storage

  • ElementForge uses browser local storage to save your preferences and settings
  • Extracted element data is only temporarily stored in memory while you're using the extension
  • No data is stored on external servers

Third-Party Services

ElementForge uses the following external resources for its user interface only:

  • Font Awesome (from cdnjs.cloudflare.com): Used for icons in the extension interface
  • Google Fonts: Used for typography in the extension interface

These resources are only used to render the extension's own interface and are not used to process any user data.

User Control and Rights

You have complete control over:

  • Which elements you select for extraction
  • When the extension is activated
  • What settings are used for extraction
  • Clearing your saved preferences through browser settings

Changes to This Policy

We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date.
